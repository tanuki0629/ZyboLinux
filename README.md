# ZyboLinux
zybo用Linuxイメージです。
このリポジトリには次のファイルが格納されています。

## 格納ファイル
- **TopZybo.bit** <br>
・FPGAデータが格納されているファイル。LED(LD0)が1s周期で点滅する。
- **zynq-zybo.dtb** <br>
・ハードウェア情報やPS・PL設定の情報が格納されているファイル
- **boot.bin** <br>
　・ブートローダを起動するための設定情報が格納されているファイル。
- **u-boot.img** <br>
　・U-Bootファイル
- **uENv.txt** <br>
　・u-boot用の環境変数を定義しているファイル
- **zImage** <br>
　・Linux カーネルファイル
